Sample Module

Introduction

In order to modify the online help as an Equipment Module developer, you have to modify the "OnlineHelp.html" files that are contained in the "OnlineHelp.zip" file of the project.

Help for First Configuration Page

In OnlineHelpTopics.xml you can specify Topics for a context sensitive online help. If the name of a Topic is identical to the name of the currently displayed page of the Editor, the MAC automatically navigates to the Anchor of the Topic.
